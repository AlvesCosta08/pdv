# Sistema Web PDV (Ponto de Venda)

## Descrição

O Sistema Web PDV é uma aplicação desenvolvida em PHP para gerenciar operações de ponto de venda em um ambiente de loja. Este sistema é projetado para simplificar o processo de vendas, controle de inventário e gestão financeira, proporcionando uma interface intuitiva e recursos avançados para melhorar a eficiência das operações comerciais.

## Funcionalidades

- **Gestão de Vendas**
  - Processamento de vendas e emissão de recibos.
  - Cadastro de produtos e serviços.
  - Aplicação de descontos e cupons.

- **Controle de Inventário**
  - Cadastro e monitoramento de produtos em estoque.
  - Atualização automática do estoque após vendas.
  - Alertas para baixo nível de estoque.

- **Gestão de Clientes**
  - Cadastro e edição de informações dos clientes.
  - Histórico de compras e informações de fidelidade.

- **Relatórios**
  - Relatórios de vendas diárias, mensais e anuais.
  - Relatórios de inventário e movimentação de produtos.
  - Relatórios financeiros e de desempenho.

- **Autenticação e Controle de Acesso**
  - Sistema de login para funcionários e administradores.
  - Controle de permissões para diferentes níveis de acesso.

## Tecnologias Utilizadas

- PHP
- MySQL
- HTML/CSS
- JavaScript
- Bootstrap (para o design responsivo)

## Requisitos

- Servidor web com suporte a PHP e MySQL (como Apache ou Nginx).
- PHP 7.4 ou superior.
- MySQL 5.7 ou superior.
